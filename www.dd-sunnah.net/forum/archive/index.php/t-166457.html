кЧЦсЩ нэ ЧснсцЬЩ Ъгуэ уцсцЯху ЧсЭЯэЫ ШЧгу " куб ЧснЧбцо "

гуь цЧсЯ УЭЯ ЧсуцсцЯэф Чсэцу нэ угЪднь ЧснсцЬЩ цсЯх ШЧгу " куб ЧснЧбцо " ЪжЧуфЧ ук ЭусЩ фебЩ гэЯфЧ ЧснЧбцо куб Шф ЧсЮиЧШ бжэ Чссх кфх.

цапб цЧсЯ куб ЧснЧбцо см "ЭбЧп" : " Уффэ Удбн УШфэ ШЧгу куб ЧснЧбцо фебЩ сгэЯфЧ куб Шф ЧсЮиЧШ цбЯЧ ксь ЧсвцЧб Чсаэф ЪиЧцсцЧ ксэх нэ ЧсУкйуэЩ цлэбхЧ ".

пуЧ Якь Ьуэк ЧсТШЧС цЧсдШЧШ Уф эфебцЧ ЧснЧбцо куб Шф ЧсЮиЧШ Шпс уЧ эгЪиэкцф.
эапб Уф ЧснсцЬЩ ЪдхЯ ЭусЩ дШЧШэЩ фебЩ скуб ЧснЧбцо цбЯЧ ксь уф УгЧС Хсэх нэ Чсксф цШЭуЧэЩ ЧсоцЧЪ ЧсУуфэЩ. ( ЭбЧп )
