السلام عليكم

اخي ابن خلف

اليك ترجمة القصيدة باللغة العربية ومعناها والمقصود بها :

(العبارات بين الاقواس هي لشرح معاني الكلمات وعبارات القصيدة الشعبية العراقية ) :

اخرج من ديارك ضلوا يكولون

(اخرج من بيتك ما زالوا يقولون )

الامم نفس الامم والزمن يتكرر

(الامم هي نفسها والزمن يعاد بوقائعه )

لا طاقة لنا اليوم بجالوت
اذهب قاتل انت وربك الاكبر

(اشارة للاية الكريمة وان اليهود اعترضوا على القتال مع موسى )

لو اخرتنا انقاتل بعد جم يوم
(اي لو اخرت قتالنا بعد كم يوم )
اذا رايد تقاتل ليش نتأخر ؟

(تسائل : طالما تريد القتال فلما التاخير ؟؟ )
كتاب المهدي يظهر ؟
( تسائل اخر : اي تاخير القتال انتظارا لكتاب من القائم الحجة المهدي ؟ )


وانت ميت خوف ( وانت من الخوف وكانك ميت بلا حراك )

وسكتت ولحم اخوك ابابك مطشر
(سكت عن مقتل اخوتك امام دارك اي عينيك ولحومهم مبعثرة امام دارك )
كتاب المهدي المن ؟
( سؤال : كتاب المهدي لمن ؟ )
حتى ضده اتصير
(اي حتى تصبح ضد المهدي ؟ )
ما دام بكتاب المهدي
( طالما بكتاب المهدي ) (اي طالما تنتظر كتاب المهدي )
ما يظهر (فلن يظهر ابدا )

مو انت المتاني
(اي ليس انت المنتظر له – اشارة الى الكتاب - )
المهدي الي امتانيك
(المهدي هو الذي ينتظرك تتحرك )
اذا حاضر يشوفك والحسن يحضر
(اذا كنت مستعد فهو يراك وحتى الامام الحسن معك يحضر )

من ذاتك تجرد تلك ذاته وياك
(اي تجرد من الانا الشخصية تجد ان ذات المهدي معاك اي بعينه معاك )
ومن خوفك تحرر حتى تتحرر
(اي تخلص من خوفك تحصل على حريتك )

يالرايح بذات ربك على الفضائيات
(اي : احلفك بذات الله يا الذاهب الى القنواة الفضائية )

وحبلن بالسجون وجابن المنكر (المذكر )
( اشارة للمعتقلات العراقيات في سجون الاحتلال واغتصابهن وانهن ولدن هناك المنكر او المذكور من الاطفال )

ما هزك ضميرك انت من يا دين
( اشارة لهذه الفعلة : الم يهتز ضميرك لها ؟ انت من اي دين ؟ اي ماهو دينك ؟ نوع من الاستنكار )

مو حتى اليهود الشافهة استنكر
(مو : حرف استفهام وتوكيد واثبات للسامع )
(حتى اليهود الذين راوا الفعلة الشنيعة استنكروها -مبالغة على الفعلة الشنيعة حتى الذين يعملون الفضائع لايقبلون مثل هذه الفعلة )

وفسح للنجف سني
(ربما يعني ان حتى السني يدافع عن النجف وانت لا )

وانت ما حسيت
(وانت لم تحس بدفاع السني هذا للنجف )

بعد لا تكول شيعي
(اي لاتقول بعد هذا انك شيعي )

اشجابك الحيدر ؟
( اين انتمائك لعلي بن ابي طالب ؟ استنكار اخر وتحذير بان تقول انك منتمي لعلي بن ابي طالب )

اصبابة عليك
( اعتقد معنى صبابة : هو القدسية والعظمة والتنزيه عن كل خطأ او محاسبة )

وما فتحت تخاف
(اي بعد هذه القدسية وانت تعلن الخوف ؟ )

شنخالة علي
( اي نبقى ننخي (النخوة : هي الفزعة عند العراقيين : وهي الدعوة للنصرة بالعربية الفصحى – اي نبقى ندعو الى نصرة علي بن ابي طالب ( ويسمى بالعراقي حيدر الكرار )


جا بيتكم خيبر ؟

(جا : بالجيم المعجمة تحتها ثلاثة نقاط . باللهجة العراقية تعني الاستفهام : هل )
(بيتكم خيبر : يعني محصن كما كانت قرية خيبر اليهودية لايفتحها الا الامام علي برفع بابها الثقيل جدا ؟ )

واليك القصيدة بالعربية الفصحى :

اخرج من ديارك ما زالوا يقولون
الامم نفس الامم والزمن يتكرر

لا طاقة لنا اليوم بجالوت
اذهب قاتل انت وربك الاكبر

لو اخرتنا ان نقاتل بعد كم يوم

اذا تريد القتال لماذا نتأخر ؟

كتاب المهدي يظهر ؟
وانت من الخوف ميت
وتسكت ولحم اخوك امام بابك مبعثر؟

كتاب المهدي لمن ؟
حتى ضده تصير ؟
ما دام بكتاب المهدي اي القتال

فلن يظهر هذا الكتاب .

ليس انت المنتظر للكتاب
المهدي هو الذي ينتظرك
اذا كنت جاهز يشوفك والحسن كذلك يحضر


من ذاتك تجرد تجد ذاته معك

ومن خوفك تحرروتخلص كي تتحرر

يالذاهب الى الفضائيات احلفك بذات ربك

وحملن داخل السجون من الاغتصاب و ولدن المنكر
الم يهتز ضميرك ؟ انت من اي دين ؟

الا حتى اليهود الذين راوا الفعلة استنكرها

وفسح للنجف اهل السنة يدافعون عنها

وانت لم تحس بعد
بعد هذا لاتقول انا شيعي
لاتقترب الى علي بن ابي طالب
هل عليك القدسية والتنزيه ؟
ولا زلت تخاف
هل تبقى تدعو نصرة علي ؟
فهل بيتكم خيبر ؟؟
